## 📌 Project Overview
This project demonstrates a rigorous implementation of **Linear Regression from scratch** using only **NumPy**. The goal is to compare different model specifications through two distinct lenses: **Numerical (Gradient Descent)** and **Statistical (Normal Equation)**, while incorporating **Lasso (L1)** and **Ridge (L2)** regularization to handle overfitting and feature selection.
